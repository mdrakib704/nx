# NX Panel

NX Panel is a lightweight, ultra-performant, open-source hosting panel designed to run smoothly on environments with as little as 2 GB RAM. It uses 40 MB–80 MB idle RAM.

## Features
- **Multi-Environment Hosting**: Minecraft Java/Bedrock, Discord.js, Python, Node.js, PHP, Static Sites, Custom Apps.
- **Low Footprint**: Built with Node.js, Fastify, and SQLite. No heavy dependencies.
- **Coin System**: Built-in economy, daily rewards, referrals, and a resource upgrade shop.
- **Background Manager**: Admins can upload MP4/WebM backgrounds dynamically.
- **Ad Manager**: Schedule banners, popups, and video ads with rewards.
- **Real-Time**: WebSockets for live console, stats, and notifications.
